# 紫微斗数全集·节选
# 清代古本，内容侧重各宫各星具体论断

== 十二宫诸星得地合格诀 ==

命宫得紫微：主一生有权威，气宇轩昂，贵人多助。
命宫得天府：主一生衣食丰足，财源稳定，处世稳重。
命宫得太阳（入庙）：主光明正大，官途显达，名誉佳。
命宫得太阴（入庙）：主富贵，尤利女命，感情细腻。
命宫得武曲：主财赋，性刚，宜经商或武职。
命宫得天同：主福寿，性情温和，一生少灾。
命宫得天机：主聪明，谋略佳，但多变。
命宫得廉贞（清白）：主官禄，有才干，情绪起伏。
命宫得贪狼：主多才，桃花重，欲望强，宜艺术业。
命宫得巨门：主口才，是非多，宜从事口才相关行业。
命宫得天相：主正直，衣食不缺，性情厚道。
命宫得天梁：主长寿，逢凶化吉，有宗教缘。
命宫得七杀：主刚毅，宜军警，年轻多磨难，中年后成。
命宫得破军：主开创，一生变动多，宜创业。

== 十二宫诸星失陷破格诀 ==

命宫太阳落陷：劳心费力，名誉受损，贵人难遇。
命宫太阴落陷：暗耗多，情绪起伏，女命婚姻多波折。
命宫廉贞见煞：官非牢狱，血光，感情纠纷。
命宫七杀见羊陀火铃：刑伤，意外，事业多败。
命宫破军见煞：破财破家，感情多变，漂泊不定。
命宫贪狼见空劫：宜出家，或从事宗教玄学，否则财散。

== 财帛宫论 ==

财帛宫主财源之多寡、财运之好坏。

财帛宫得武曲化禄：正财旺，经商有道，财源不断。
财帛宫得太阴化禄：不动产收益丰，偏财亦旺。
财帛宫得天府：财库充实，善守财，积累稳健。
财帛宫得禄存：一生财源不缺，但守财性强，吝于施舍。
财帛宫得化忌：财来财去，破财，须防投资失利。
财帛宫得空劫：财务虚耗，不宜投资，宜稳健理财。
财帛宫得羊陀：财务受阻，横财难守，须防血财。

== 官禄宫论 ==

官禄宫主事业、工作、官途之吉凶。

官禄宫得紫微化权：官位显赫，独揽大权，事业顶峰。
官禄宫得太阳入庙：官途光明，名誉佳，上司提携。
官禄宫得武曲化权：事业主动，财务掌权，适合经营管理。
官禄宫得化禄：事业顺遂，财官双收，贵人助力。
官禄宫得化忌：事业受挫，小人多，升迁受阻，须防职场是非。
官禄宫得七杀：宜武职，创业，不宜久任文职。
官禄宫得破军：事业多变，跳槽频繁，宜创业非宜打工。
官禄宫得空劫：事业难成，宜务虚行业（文艺、宗教、顾问）。

== 夫妻宫论 ==

夫妻宫主配偶之气质、感情之吉凶、婚姻之迟早。

夫妻宫得紫微：配偶气质非凡，有领导力，但主宰欲强。
夫妻宫得天府：配偶稳重守财，婚姻安稳。
夫妻宫得太阴入庙：配偶（尤其妻子）贤淑，感情细腻。
夫妻宫得贪狼：感情复杂，桃花多，婚前多感情波折。
夫妻宫得廉贞化忌：婚姻波折，感情是非，须防外遇。
夫妻宫得化忌：婚姻多波折，感情不顺，易有离婚之虞。
夫妻宫得七杀：配偶个性刚烈，相处需磨合，宜晚婚。
夫妻宫得破军：婚姻多变，离婚率高，感情不稳。
夫妻宫得空劫夹：感情虚耗，缘分薄，或独身。

== 迁移宫论 ==

迁移宫主外出、旅行、异乡运势，亦主人际关系与社会评价。

迁移宫得太阳入庙：外出大吉，异乡发展顺遂，人缘极佳。
迁移宫得天马：一生奔波，适合外派、移民、业务。
迁移宫得化禄：外出财运佳，异乡发展优于故乡。
迁移宫得化忌：外出不顺，旅途多险，人际纠纷多。
迁移宫得煞星：外出须防意外，人际关系复杂。
迁移宫空宫：借命宫论，外出运势随命格高下。

== 疾厄宫论 ==

疾厄宫主身体健康、疾病之所在。

疾厄宫得天同：身体佳，少病，寿元长。
疾厄宫得天梁：逢病能化解，有贵人医治，康复力强。
疾厄宫得化忌：该星所主之身体部位须特别留意。
疾厄宫得廉贞化忌：须防血光、手术、炎症。
疾厄宫得武曲化忌：须防金属伤、骨骼、手术。
疾厄宫得太阴化忌：须防泌尿、妇科、暗疾。
疾厄宫得羊陀：须防意外伤、手术、慢性病。
疾厄宫得空劫：疾病难以诊断，宜中医调理。

== 福德宫论 ==

福德宫主精神享受、内心世界、兴趣嗜好。

福德宫得天同：精神愉快，享受生活，知足常乐。
福德宫得太阴：内心细腻，重情感，喜独处。
福德宫得贪狼：兴趣广泛，享受感官，欲望多。
福德宫得化忌：精神压力大，多思多虑，内心不安。
福德宫得空劫：精神虚耗，宜修行、冥想，或从事创意工作。
福德宫得天梁：精神上倾向哲学、宗教，老来享清福。

== 田宅宫论 ==

田宅宫主不动产、家宅、财库实力。

田宅宫得天府：田产丰厚，财库充实，能守祖业。
田宅宫得太阴入庙：房产多，不动产收益丰。
田宅宫得武曲化禄：财库充实，投资房产有利。
田宅宫得化忌：家宅不安宁，搬迁多，或房产纠纷。
田宅宫得空劫：财库空虚，难置产，或有而无实。
田宅宫得破军：祖业难守，多次搬迁，宜自置。
